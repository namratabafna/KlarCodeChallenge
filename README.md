# Employee Performance Management

## Requirements
Design a web application that allows employees to submit feedback toward each 
other's performance review.
*Partial solutions are acceptable. * It is not necessary to submit a complete 
solution that implements every requirement.

### Admin view
* Add/remove/update/view employees
* Add/update/view performance reviews
* Assign employees to participate in another employee's performance review

### Employee view
* List of performance reviews requiring feedback
* Submit feedback

## Project setup

```
npm install
```

### Compiles and hot-reloads for development
